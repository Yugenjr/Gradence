# OS Daily Planner & Habits

The OS Daily Planner provides a unified workspace for managing your daily classes, habits checklist, and milestone countdowns. All logs sync reactively with the Home Dashboard.

---

## 1. Class Timetables
* **Setup Schedule**: Add class items inside the Planner. Enter the subject name, start time (e.g. `09:30 AM`), and assign a location or room number.
* **Dashboard Integration**: The home dashboard reads these timetables from the shared preferences layer and lists your lectures in chronological order.

---

## 2. Habit Checklist & Streaks
* **Checklist Setup**: Create daily habits (e.g. "Attend All Lectures", "Solve 1 Coding Challenge").
* **Dashboard Toggling**: Habits can be checked off directly from the home dashboard. The updated completion state is saved instantly to Capacitor Preferences, keeping both panels aligned.

---

## 3. Custom Countdown Targets
* **Setting Targets**: Input target event names (e.g. "Major Project Submission", "Semester Finals") and select a date.
* **Dynamic Counters**: The dashboard calculates the exact days remaining in real-time, showing active countdown cards.
